# 前言

首先需要指出的是，cfw wiki 绝对是我这几年看到过最好的越狱百科全书。完全是保姆教程，包教包会的那种。

但非常遗憾的消息是，cfw wiki 跟进本土化翻译的速度实在是太慢了————甚至我可以直接下定论，他们根本也绝对没有 review 过我的任何一条翻译条目。我在一年半之前提交了近 35000 词的翻译，然而到现在中文翻译的进度还是 84%，最近的一些条目根本没人去跟进。

其实这也和国内越狱社区日渐式微的现状息息相关：东西做出来没人看了，或者没人耐得住性子看了，那么你这个东西写起来有什么意义呢？单拿苹果越狱吧举例，没有任何一个小白能跑得过那几个所谓黄牌的洗礼的，要么私聊交钱，要么就是根本没人去理他们。国内能叫的上名的开发者，好像一只手就能数过来。

这倒也能理解，毕竟苹果一直在拿越狱社区反向吸血。越狱社区能玩的东西，什么外观增强，什么美化包，多任务，剪贴板，更好的锁屏，到后面都给苹果玩了，自然选择弃暗投明回归白苹果。余下的，要么就是开发者心力交瘁，或者是开发者被招安，或者只是不再热爱这个社区了而已，于是某一天，这个曾经热闹的社区也就随之寂静了。

不得已我还是只能自己一个人开这个 repo 了，虽然我觉得没什么人会看，但我喜欢。这是我喜欢的东西，我自然会去做。有人做我觉得总是好的，至少可以看到的是，越狱社区还有我在坚持把这些基础的问题翻译一遍，教各位新来的越狱人，无论是跟风还是真的热爱这一门技术怎么应对这些问题。

我可以说，当然，没人塞钱给我我没必要做这个，但是我喜欢，我乐意；我喜欢越狱的那种精神，突破一切人为桎梏向外生长的精神。

# 译名问题

国内由于对文档内的东西普及较少，这里我统一对一些可能混淆的地方进行说明： - Semi-Tethered / Semi-Untethered。前者指的是连接电脑才能完成整个越狱流程的越狱软件，后者指的是只需要每次重启之后再激活一下越狱。两者在国内基本上不分，都叫不完美或者半完美越狱；这里我特别拎出来说，前者叫半连接越狱，后者叫半完美越狱。 - Blob。关于这个东西标准的叫法是“二进制大型对象”，名字很长而且也很泛泛而谈，很空。对于这个你可以理解为类似 Token（特指令牌）的东西，当然，具体语境具体分析。如果没有特别指出来的话，一般把它作为存根或者令牌就行。 - 别的想到了我再继续补充。
